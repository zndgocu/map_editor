# fms 맵에디터 툴
- node를 그릴 수 있습니다.
- vertex를 그릴 수 있습니다.
- 이후 fms webClient와 통합되어야 합니다.
- 이후 signalR Client가 적용되어야 합니다.
- 이후 Blazor와 통합되어야 합니다.

# 진행사항
- node, vertex 드로잉 모드가 추가되었습니다.
- viewport 설정을 위한 open 모드가 추가되었습니다.
- 데이터 저장이 추가되었습니다.
- 데이터 열기가 추가되었습니다.
- 배경 맵을 로딩할 수 있게되었습니다.
